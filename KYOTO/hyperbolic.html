Aigner's conjectures

Aigner's conjectures

Let \(p, q\) be real non negative numbers and \(i > 0\) then

  • \(\|(q,p) \|_s < \|(q + i,p) \|_s\)
  • \(\|(q,p) \|_s < \|(q ,p +i ) \|_s\)
  • If in addition \(p < q\) then \(\|(q ,p ) \|_s < \|(q + i ,p -i ) \|_s\)
