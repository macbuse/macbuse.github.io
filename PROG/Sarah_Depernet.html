SUJET : Sudoku (résolution d’une grille, constructeur de grilles avec solution unique, estimer la difficulté d’une grille)¶

This is easy and you can find the code to do it on GitHub (I did). But what if it's in a jpg

  • automatically download it
  • recognise the digits
  • return the answer as a jpg with my handwriting

Pour ce projet, on va donc importer une grille de sudoku depuis une page web pour la rsoudre et la renvoyer remplie, sous format jpeg, avec une criture manuscrite. La mthode employe se spare en plusieurs tapes :

1) De l'image à la matrice¶

  • On importe la grille sous forme d'image grce une requte
  • On divise cette image en 81 petites images correspondant aux cases
  • On entrane le module de reconnaissance des caractres, puis on analyse les images
  • La grille traduite se prsente sous forme d'une matrice o on range les rsultats des tests

2) Résolution du sudoku¶

  • On rsout le sudoku, par back-tracking : on revient en arrire ds qu'on dtecte une erreur.

3) Remplissage de la grille avec les chiffres manuscrits¶

  • Je reprends le bout de code que vous nous aviez montr pour rcuprer votre criture
  • On "colle" chaque chiffre manuscrit sa place dans l'image de dpart selon la solution trouve au sudoku
  • On renvoie et sauvegarde l'image obtenue du sudoku rsolu

c = 46 #414/9 # ct d'une case
def decoupe(im,coordonnes):
    a, b = coordonnes
    marge = 2 # arbitrairement
    case = im[a*c+marge:(a+1)*c-marge,b*c+marge:(b+1)*c-marge]
    case = transform.resize(case, (28,28))
    case[case > .6] = 1
    negatif = 1. - case # Pour respecter le format des images de la mthode de reconnaissance
    return negatif

def absent_sur_bloc(k , grille , l , c): # Fonction qui vrifie si le chiffre 'k' est deja prsent sur le bloc
    _l = l-(l%3)
    _c = c-(c%3)
    for c in range(_c,_c+3):
        for l in range(_l,_l+3):
            if grille[l][c] == k:
                return False
    return True 
    

def est_valide(grille , position): # Fonction qui vrifie si on doit modifier la case, la case est dfinie avec le paramtre position qui s'agrmente de 1  chaque fois
    global test
    global iteration
    iteration = iteration+1
    if position == 9*9:
        return True
        
    l = position//9
    c = position%9
    
    if grille[l][c] != 0:

Avec chacun des chiffres en criture manuscrite, on va crer une vignette carre que l'on n'aura plus qu' coller la bonne place sur l'image.

def redim(im,c): # cre une vignette carre ( la taille des cases de la grille) du chiffre avec l'criture manuscrite
    n, p = len(im), len(im[0])
    ratio = p/n
    case = np.zeros(shape=(c,c))+1.       
    if n>=p :
        p_prime = int(c*ratio)
        vignette = transform.resize(im, (c,p_prime))
        marge = int((c-p_prime)/2)
        case[:,marge:marge+p_prime] = vignette
    else :
        n_prime = int(c/ratio)
        vignette = transform.resize(im, (n_prime,c))
        marge = int((c-n_prime)/2)
        case[marge:marge+n_prime,:] = vignette
    return case
