Analyse des Vecteurs Propres de la Matrice de Transition

La structure de la matrice $P$ est fondamentale pour comprendre ses vecteurs propres. Il s'agit d'une matrice stochastique par blocs, ce qui signifie que l'espace des tats $\{1, \dots, 6\}$ est divis en sous-ensembles aux comportements distincts : l'tat absorbant (1), les tats transitoires (2, 3), et la classe terminale (4, 5, 6).

1. Les Valeurs Propres ($\lambda$)

Pour toute matrice de transition, la plus grande valeur propre est toujours $\lambda = 1$. Dans votre cas :

  • $\lambda = 1$ est une valeur propre double : Cela indique la prsence de deux classes fermes (ensembles d'tats dont on ne peut plus sortir). Ici, l'tat $\{1\}$ et le bloc $C = \{4, 5, 6\}$.
  • $|\lambda| < 1$ : Les autres valeurs propres sont associes aux tats transitoires. Elles dterminent la vitesse laquelle la chane quitte les tats 2 et 3 pour tre absorbe par l'une des deux classes fermes.

2. Vecteurs Propres Gauche (Mesures Invariantes)

Les vecteurs propres gauche ($vP = \lambda v$) associs $\lambda = 1$ reprsentent les distributions de probabilit stationnaires. Comme il y a deux classes fermes, il existe deux vecteurs de base linairement indpendants :

A. Le point fixe de l'tat 1

Si la chane commence ou tombe dans l'tat 1, elle y reste indfiniment. $$v_1 = (1, 0, 0, 0, 0, 0)$$

B. La loi stationnaire du bloc $C = \{4, 5, 6\}$

Le bloc $C$ est une classe terminale irrductible et apriodique. Une fois l'intrieur, la chane circule entre 4, 5 et 6. long terme, la distribution converge vers un quilibre interne. En rsolvant le systme $\mu_C P_C = \mu_C$ avec $\sum \mu_i = 1$, on obtient :

  • $\mu_4 = 3/13 \approx 0.231$
  • $\mu_5 = 1/13 \approx 0.077$
  • $\mu_6 = 9/13 \approx 0.692$

def simuler_trajectoire(matrice, x_init, n=100):
    """
    Simule une trajectoire de chane de Markov sans utiliser append.
    
    matrice : np.array (la matrice de transition)
    x_init  : int (tat initial de 1  N)
    n       : int (nombre de sauts/transitions)
    """
    # 1. On rcupre la dimension dynamiquement
    nb_etats = matrice.shape[0]
    indices_etats = np.arange(1, nb_etats + 1)
    
    # 2. Pr-allocation du tableau (taille n + 1 pour inclure l'tat initial)
    trajectoire = np.zeros(n + 1, dtype=np.int32)
    trajectoire[0] = x_init
    
    # 3. Boucle de simulation
    for t in range(1, n + 1):
        # On utilise l'tat prcdent (t-1) pour calculer l'tat actuel (t)
        trajectoire[t] = np.random.choice(
            indices_etats, 
            p=matrice[trajectoire[t-1] - 1]
        )
        
    return trajectoire

# Exemple d'excution

import numpy as np

# Matrice de transition (P)
P = np.array([
    [1,    0,    0,    0,    0,    0],
    [1/4, 1/2, 1/4, 0,    0,    0],
    [0,   1/5, 2/5, 1/5, 0,   1/5],
    [0,   0,   0,   1/6, 1/3, 1/2],
    [0,   0,   0,   1/2, 0,   1/2],
    [0,   0,   0,   1/4, 0,   3/4]
])

def estimer_absorption(x_depart, 
                       n_sim=1000, 
                       n_pas=100):
    nb_etats = P.shape[0]
    etats = np.arange(1, nb_etats + 1)
    
    resultats_finaux = []
    
    for _ in range(n_sim):
        courant = x_depart
        for _ in range(n_pas):
            # Tirage de l'tat suivant
            courant = np.random.choice(etats, p=P[courant - 1])
            
            # Optimisation : si on atteint une classe terminale, on s'arrte
            if courant == 1 or courant >= 4:
                break
        resultats_finaux.append(courant)
    
    resultats_finaux = np.array(resultats_finaux)
    
    # Calcul des proportions
    prob_1 = np.mean(resultats_finaux == 1)
    prob_C = np.mean(resultats_finaux >= 4)
    
    return prob_1, prob_C

# Excution pour les deux points de dpart

3. Comparaison avec les valeurs thoriques

Pour valider les simulations, on utilise le calcul thorique des probabilits d'absorption. Soit $h_i$ la probabilit d'tre absorb par l'tat 1 en partant de l'tat $i$. Le systme d'quations se base sur la dcomposition au premier pas :

  1. $h_1 = 1$ : L'tat 1 est absorbant.
  2. $h_2 = \frac{1}{4}h_1 + \frac{1}{2}h_2 + \frac{1}{4}h_3$ : Depuis 2, on va en 1 (prob. 1/4), on reste en 2 (prob. 1/2) ou on va en 3 (prob. 1/4).
  3. $h_3 = \frac{1}{5}h_2 + \frac{2}{5}h_3 + \frac{2}{5}(0)$ : Depuis 3, on va en 2 (prob. 1/5), on reste en 3 (prob. 2/5) ou on entre dans le bloc $C$ (prob. 1/5 + 1/5 = 2/5). Si on entre dans $C$, la probabilit d'atteindre 1 devient nulle.

En rsolvant ce systme :

  • De (3), on tire : $\frac{3}{5}h_3 = \frac{1}{5}h_2 \implies h_3 = \frac{1}{3}h_2$.
  • En remplaant dans (2) : $\frac{1}{2}h_2 = \frac{1}{4} + \frac{1}{4}(\frac{1}{3}h_2) \implies \frac{1}{2}h_2 - \frac{1}{12}h_2 = \frac{1}{4} \implies \frac{5}{12}h_2 = \frac{3}{12} \implies h_2 = \frac{3}{5}$.
