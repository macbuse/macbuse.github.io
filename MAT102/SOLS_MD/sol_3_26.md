## **Exercice 23 a)** üá´üá∑
Points : $A = (1, 0, 1)$, $B = (1, 1, 0)$, $C = (1, 0, 0)$.

### **1. Vecteurs Directeurs**
$$\vec{AB} = B - A = (1-1, 1-0, 0-1) = (0, 1, -1)$$
$$\vec{AC} = C - A = (1-1, 0-0, 0-1) = (0, 0, -1)$$

### **2. Vecteur Normal**
$$\vec{n} = \vec{AB} \times \vec{AC} = \begin{pmatrix} 0 \\ 1 \\ -1 \end{pmatrix} \times \begin{pmatrix} 0 \\ 0 \\ -1 \end{pmatrix} = (-1, 0, 0)$$
Les coefficients sont $a=-1$, $b=0$, $c=0$.

### **3. √âquation Cart√©sienne**
L'√©quation est $-1x + 0y + 0z + d = 0$, soit **$-x + d = 0$**.

$$-1(1) + d = 0 \implies d = 1$$

L'√©quation cart√©sienne du plan $\mathcal{P}$ est :
$$\mathcal{P}: -x + 1 = 0 \quad \text{ou} \quad \mathbf{x = 1}$$

## **Exercice 23 b)** üá´üá∑
Points : $A = (2, 0, 1)$, $B = (1, 1, 1)$, $C = (-1, 0, -1)$.

### **1. Vecteurs Directeurs**
$$\vec{AB} = B - A = (1-2, 1-0, 1-1) = (-1, 1, 0)$$

$$-1 + d = 0 \implies d = 1$$
