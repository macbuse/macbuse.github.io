Correction

Nous voulons montrer que les solutions de l’équation suivante sont un cercle dans le plan complexe :

Re(z+2iz−1)=0\text{Re} \left( \frac{z + 2i}{z - 1} \right) = 0

où z=x+iyz = x + iy, avec xx et yy les parties réelle et imaginaire de zz.

Étape 1 : Partie réelle de l’expression

D’après les calculs pour la partie a) a partie réelle de l’expression totale est :

Re(z+2iz−1)=x(x−1)−y2−2y(x−1)2+y2\text{Re} \left( \frac{z + 2i}{z - 1} \right) = \frac{x(x - 1) - y^2 - 2y}{(x - 1)^2 + y^2}

Nous devons résoudre l’équation suivante :

x(x−1)−y2−2y=0x(x - 1) - y^2 - 2y = 0

Étape 2 : Résolution de l’équation

Développons l’équation :

x2−x−y2−2y=0x^2 - x - y^2 - 2y = 0

Réarrangeons les termes pour obtenir une forme plus familière :

x2−x=y2+2yx^2 - x = y^2 + 2y

Ajoutons 14\frac{1}{4} des deux côtés pour compléter les carrés, à gauche pour le terme en xx et à droite pour le terme en yy :

x2−x+14=y2+2y+1x^2 - x + \frac{1}{4} = y^2 + 2y + 1

Ce qui donne :

(x−12)2=(y+1)2\left( x - \frac{1}{2} \right)^2 = (y + 1)^2

Étape 3 : Interprétation géométrique

Nous avons l’équation suivante :

(x−12)2+(y+1)2=(12)2\left( x - \frac{1}{2} \right)^2 + (y + 1)^2 = \left( \frac{1}{2} \right)^2

Il s’agit de l’équation d’un cercle de centre (12,−1)\left( \frac{1}{2}, -1 \right) et de rayon 12\frac{1}{2}.

Conclusion

Les solutions de l’équation Re(z+2iz−1)=0\text{Re} \left( \frac{z + 2i}{z - 1} \right) = 0 forment un cercle dans le plan complexe, de centre (12,−1)\left( \frac{1}{2}, -1 \right) et de rayon 12\frac{1}{2}.
