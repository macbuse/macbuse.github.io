To explain that the 2-sphere (S2S^2) is a manifold using stereographic projection, you are essentially showing that the sphere can be covered by “flat” coordinate maps (charts). This is the perfect example of the “Differentiable Viewpoint” because it uses smooth functions to link the curved geometry of the sphere to the flat plane of ℝ2\mathbb{R}^2.

1. The Setup

We define the unit 2-sphere in ℝ3\mathbb{R}^3 as: S2={(x,y,z)∈ℝ3:x2+y2+z2=1}S^2 = \{(x, y, z) \in \mathbb{R}^3 : x^2 + y^2 + z^2 = 1\}

To prove it is a manifold, we need to find a collection of open sets that cover the sphere and “map” them smoothly to ℝ2\mathbb{R}^2. We use two specific points to project from: 1. The North Pole (NN): (0,0,1)(0, 0, 1) 2. The South Pole (SS): (0,0,−1)(0, 0, -1)

2. The Projection Formula (The “North” Chart)

Imagine a light source at the North Pole. For any point P(x,y,z)P(x, y, z) on the sphere (except the North Pole itself), we draw a line from NN through PP until it hits the xyxy-plane (z=0z=0).

The point where it hits the plane is U(u,v)U(u, v). Using similar triangles or line equations, the mapping σN:S2\{N}→ℝ2\sigma_N: S^2 \setminus \{N\} \to \mathbb{R}^2 is: u=x1−z,v=y1−zu = \frac{x}{1-z}, \quad v = \frac{y}{1-z}

Why this matters:

  • Smoothness: This map is smooth (infinitely differentiable) everywhere except at z=1z=1 (the North Pole).
  • Homeomorphism: It is a one-to-one mapping between the “punctured” sphere and the entire flat plane.

3. Covering the Whole Sphere

One map isn’t enough because we “missed” the North Pole. To fix this, we create a second map, σS\sigma_S, by projecting from the South Pole onto the plane. σS:S2\{S}→ℝ2\sigma_S: S^2 \setminus \{S\} \to \mathbb{R}^2 u′=x1+z,v′=y1+zu' = \frac{x}{1+z}, \quad v' = \frac{y}{1+z}

Now, every single point on the sphere is covered by at least one of these two open “neighborhoods” * UN=S2\{N}U_N = S^2 \setminus \{N\} * US=S2\{S}U_S = S^2 \setminus \{S\}

4. The “Differentiable” Check (Transition Maps)

In Milnor’s view, for this to be a differentiable manifold, the overlap between these two maps must be smooth. If you take a point in the overlap (everywhere except the poles), map it to the plane via σN\sigma_N, and then map it back to the other plane via σS\sigma_S, the result must be a smooth function.

The transition map σS∘σN−1\sigma_S \circ \sigma_N^{-1} from ℝ2\{(0,0)}\mathbb{R}^2 \setminus \{(0,0)\} to itself is: (u,v)↦(uu2+v2,vu2+v2)(u, v) \mapsto \left( \frac{u}{u^2 + v^2}, \frac{v}{u^2 + v^2} \right) Since u2+v2≠0u^2 + v^2 \neq 0 on the overlap, this function is smooth.

Summary for your explanation:

  • The Atlas: You have two charts (σN,σS)(\sigma_N, \sigma_S).
  • The Manifold: Because these charts cover S2S^2 and the transition between them is smooth, S2S^2 is a smooth 2-manifold.
  • Visual Analogy: It’s like a world map. You can’t represent the whole Earth on one flat map without a “tear,” but you can use two maps (like two hemispheres) that overlap smoothly at the edges.
